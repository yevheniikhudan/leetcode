# Tutorial: Palindrome Partitioning (LeetCode 131)

## Key Concepts
- Backtracking
- Palindrome checking
- Recursive partitioning

## Approach Outline
- Use backtracking to explore all possible partitions
- At each step, check if the current substring is a palindrome
- If so, recursively partition the rest of the string
- Collect all valid partitions in the result
