# 371. Sum of Two Integers

**Difficulty:** Medium

**Topics:** Math, Bit Manipulation

---

## Problem Description

Given two integers `a` and `b`, return their sum without using the operators `+` and `-`.

---

## Examples

**Example 1:**
```
Input:  a = 1, b = 2
Output: 3
```

**Example 2:**
```
Input:  a = 2, b = 3
Output: 5
```

**Example 3:**
```
Input:  a = -1, b = 1
Output: 0
```

---

## Constraints

- `-1000 <= a, b <= 1000`
