# Tutorial: N-Queens (LeetCode 51)

## Key Concepts
- Backtracking
- Row, column, and diagonal constraints
- Board representation

## Approach Outline
- Use backtracking to try placing a queen in each row
- Track columns and diagonals that are already occupied
- When a valid configuration is found, add it to the result
- Return all valid board configurations
